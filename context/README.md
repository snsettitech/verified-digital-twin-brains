# Context Toolkit

This folder holds templates and guidance for small, reusable context packs.

## Templates
- `context-pack.md`
- `change-brief.md`
- `decision-log.md`

## Automation
- Generate a pack: `python scripts/context_pack.py`
- Generate a map: `python scripts/repo_map.py`

Output artifacts:
- `artifacts/context/context_pack.md`
- `artifacts/context/repo_map.txt`
